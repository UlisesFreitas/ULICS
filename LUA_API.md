## Quick Reference

All 40+ Lua functions are fully documented in `docs/LUA_API.md`, including:

- ✅ Graphics (clear, pset, line, rect, circ, print, camera, etc.)
- ✅ Input (btn, btnp, mouse, mousex, mousey)
- ✅ Sprites & Maps (spr, sspr, map, mget, mset)
- ✅ **Audio** (sfx, music) - **NOW WORKING in v1.1.0!**
- ✅ Math (sin, cos, atan2, sqrt, flr, ceil, rnd)
- ✅ System (time, reset, exit, load_cartridge, goto_menu)

**New in v1.1.0:**
- `sfx(id, channel)` - Working SFX synthesis!
- `flr(x)` - **IMPORTANT:** Use this on coordinates before rendering!
